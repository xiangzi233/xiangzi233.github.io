1 Vue介绍

Vue 是用于构建用户界面的渐进式框架，其主要特点是易用、灵活、性能优，同时对于开发者而言，在有 HTML、CSS、JavaScript 基础的前提下非常容易上手。

  • Vue配置

    • 安装node.js(用到npm)

      node.js官网下载安装

      检测是否安装成功 cmd执行 node -v ,npm -v

    • 安装Vue CLI（脚手架）

      cmd执行指令

  • Vue工程目录

    ①：assets：存放项目中需要用到的资源文件，css、js、images 等。

    ②：componets: 存放 vue 开发中一些公共组件：例如项目初始的 header.vue、footer.vue 就是公共组件。

    ③：router：vue 路由的配置文件。

    ④：views：存放页面文件

    ⑤：app.vue：根

2 Vue双向绑定

Vue代码结构

每一个 Vue 文件由三部分组成，template、script、style ，它们分别对应 HTML、JavaScript、CSS。

在 template 里面只允许有一个块状元素,只能有一个快标签，其他的块标签都在这一个内，通常情况下在 template 里面写的都是 div。
