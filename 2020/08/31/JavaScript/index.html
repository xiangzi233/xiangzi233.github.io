1. 认识Javascript

1.1 JavaScript书写位置

  • 内部：在body内部并在末尾 <script>console.log("Hello World");</script>

  • 外部：添加引入方式，也在body内部且末尾

1.2 注释

1.3 字符串

加了英文单/双引号的就是字符串，数字不加引号就是数字。

1.4 模板字符串

  • 反引号：将字符串和变量包起来 `大家好，我姓${firstName}，名${lastName}`

  • 占位符：在字符串中插入变量

  • 多行字符串的话，要换行的地方直接回车就好（不需要使用、/n）

  • 利用占位符拼接表达式：

    • let number1 = 20;
let number2 = 10;
console.log(`两个数的和是：${number1 + number2} 
两个数的差是：${number1 - number2} 。`);
<!--￼2-->

    • 注意：

      • 不能重复声明同名变量
      • 变量的定义了要在使用之前
      • 能够被多次重新赋值
      • 可以不赋初始值

  • const定义变量（与let大致相同）

    • 不同点：
      • 定义时必须赋初始值
      • 只能赋值一次

2.2 数值类型

  • 十六进制

    • let number1 = 0x010; // 十六进制的16
let number2 = 0x11; // 十六进制的17
let number3 = 0x12; // 十六进制的18
<!--￼3-->

  • 极大或极小的数(用e表示)

  • 特别注意：

    • 浮点数精度丢失

    • NaN（Not a Number）非数值

      两个变量执行运算后返回的结果还是数字类型，但是执行数学运算没成功（比如 0/0 ； 字符串乘以数字 ；NaN和任何数运算）

2.3 类型转换/字符串拼接

  • 隐式转换

    • 数字字符串 加 数字,数字隐式转换为字符串

    • 数字字符串与数字字符串做非加法运算,隐式转换为数字

  • 强制类型转换

    • parseInt

      整数字符串->整数

      小数字符串->整数

      小数
